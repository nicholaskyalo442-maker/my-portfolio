# Insurance Risk Model

## 📊 Overview
This project models insurance risk using probabilistic methods to estimate expected losses and understand financial risk exposure.

## 🎯 Objective
To simulate claim frequency and severity in order to estimate total insurance losses and assess risk variability.

## 🛠 Methodology
- Claim frequency modeled using Poisson distribution
- Claim severity modeled using Exponential distribution
- Total loss = frequency × severity

## 📦 Tech Stack
- Python
- NumPy
- Matplotlib

## 📈 Key Insights
- Insurance losses are highly skewed
- A small number of claims contribute disproportionately to total loss
- Risk is driven by extreme value events

## 📊 Outputs
- Risk distribution histogram
- Expected loss calculation
- Variance and standard deviation of claims

## 💼 Business Value
Helps insurers:
- Price premiums accurately
- Manage capital reserves
- Understand tail risk exposure
